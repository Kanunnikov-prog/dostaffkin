# Dostaffkin — Доставка посылок

Приложение для оформления и отслеживания доставки посылок с визуализацией маршрута на карте.

## Функционал

- **Оформление доставки**
  - Ввод адресов с автодополнением (Яндекс Suggest)
  - Визуализация маршрута на карте
  - Выбор размера посылки (6 вариантов: XS → MAX)
  - Выбор скорости доставки (Обычная / Приоритетная)
  - Расчет стоимости с учетом тарифа и расстояния
  - Отправка заявки на сервер

- **Отслеживание**
  - Поиск по номеру отправления
  - Отображение маршрута и статусов

## Технологии

- **Frontend:** Angular 19 (standalone components)
- **Карты:** Яндекс.Карты API + OSRM (маршрутизация)
- **Стили:** CSS, адаптивная верстка
- **HTTP:** HttpClient

## Эндпоинты API

| Метод | URL | Назначение |
|-------|-----|------------|
| POST | `https://testologia.ru/delivery/create` | Создание заявки |
| GET  | `https://testologia.ru/delivery/info?id={id}` | Получение статуса |
